## 🎓 面试准备

### 技术问题
1. **RAG的工作原理**：检索文档 → 拼接prompt → LLM生成
2. **Embedding是什么**：把文字转换为向量，语义相近的向量也相近
3. **如何降低幻觉**：RAG提供依据、降低温度、优化Prompt、幻觉检测
4. **Temperature的作用**：控制随机性，0=确定，1=随机
5. **为什么用FAISS**：快速向量检索，毫秒级响应

### 产品问题
1. **AI产品的核心指标**：采纳率、准确率、拒答率、成本、延迟
2. **如何做A/B测试**：Prompt版本管理 → 同一测试集对比 → 选优
3. **如何评估效果**：离线评测（标准测试集）+ 在线评测（真实流量）
4. **如何控制成本**：限制max_tokens、缓存、选择合适模型
5. **数据驱动决策**：记录完整数据 → 计算指标 → 发现问题 → 优化迭代
