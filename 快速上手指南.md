# AI 产品经理学习平台 - 快速上手指南

## 🎯 5 分钟快速开始

### 第 1 步：安装依赖（约 2-3 分钟）

```bash
pip install -r requirements.txt
```

**国内用户加速**（推荐）：
```bash
pip install -i https://mirrors.aliyun.com/pypi/simple/ -r requirements.txt
```

### 第 2 步：配置 API Key（1 分钟）

1. 复制 `.env.example` 为 `.env`：
```bash
cp .env.example .env
```

2. 编辑 `.env` 文件，填入你的 API 密钥：
```
LLM_API_KEY=你的-api-key
```

**API 申请地址**：
- [MiniMax](https://api.minimax.chat/) - 新用户有免费额度（推荐）
- [DeepSeek](https://platform.deepseek.com/) - 价格最低
- [通义千问](https://dashscope.aliyun.com/) - 阿里云

### 第 3 步：启动应用（1 分钟）

**Windows**：
```bash
.\start.bat
```

**Mac/Linux**：
```bash
./start.sh
```

**或者直接运行**：
```bash
streamlit run main.py
```

浏览器会自动打开 `http://localhost:8501`

## 📝 常见问题

### Q1：Embedding 模型下载很慢
A：首次会下载 118MB，国内可能较慢。可以：
1. 设置镜像：`export HF_ENDPOINT=https://hf-mirror.com`
2. 或者耐心等待（只需下载一次）

### Q2：没有 API Key 怎么办
A：必须有 API Key 才能运行。推荐申请 MiniMax（有免费额度）或 DeepSeek（价格最低）。

### Q3：如何添加自己的知识库
A：在 `data/knowledge_base/` 目录下添加 .txt 文件，重启应用即可。

### Q4：成本大概多少
A：学习阶段（50-100 次交互）约 ¥5-20，使用 DeepSeek 可降低 50% 成本。

### Q5：能在手机上用吗
A：可以。启动后访问显示的 Network URL（如 `http://192.168.1.100:8501`）。

## 🚨 遇到问题

### 问题 1：端口被占用
```
OSError: [Errno 48] Address already in use
```

**解决**：
```bash
streamlit run main.py --server.port 8502
```

### 问题 2：中文乱码
**解决**：Streamlit 网页端显示正常，忽略命令行乱码。

### 问题 3：API 调用失败
```
openai.APIConnectionError
```

**检查**：
1. `.env` 文件中的 API Key 是否正确
2. 网络是否正常
3. API 额度是否用完
